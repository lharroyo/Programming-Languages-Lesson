# Programming Languages Lesson

A comprehensive collection of tutorials, documentation, and code examples for learning programming languages.

## 📚 Languages Covered

- **Python** - General-purpose, beginner-friendly language
- **Java** - Object-oriented, enterprise-level language
- **JavaScript** - Web development and beyond

## 🚀 Getting Started

1. Choose a language folder
2. Start with the `tutorials/` section for theory and concepts
3. Practice with `exercises/`
4. Review `examples/` for real-world code samples
